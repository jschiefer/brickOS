Downloading brickOS

We offer downloads from the files section of our project web-site. This provides us and you with automatic counting of the number of downloads of each of our files.

Source Tar balls (*.tar.gz)

We offer two forms of downloads:  Formal Releases and Test Releases.

These are both available from the brickOS project web - files section.

NOTE: we do not yet have an Formal BrickOS Release.  For the time being, (while we are preparing the first release under this new name) you may find the prior legOS Formal Release (legOS 0.2.6) at legOS project page - files section.

Binaries

We are working to develop the latest set of binary downloads.

Meanwhile you can refer to our former project website for some of these. You can find some install docs referred to from our Docs page.

Prepackaged versions of legOS are available for Debian GNU/Linux. See: The Package page for legos
