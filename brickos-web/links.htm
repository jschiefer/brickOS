University use of brickOS (formerly legOS)

brickOS is used in some way in each of the following efforts/courses.

  • CS148: Building Intelligent Robots, Computer Science Department, Brown University

  • Dat4 Project Site -  design and implementation of a prioritized interrupt system for legOS 0.2.3, Dept. of Computer Science, Aalborg University

  • LegOS Research a paper by Ryan Simmons, CIS 690, Kansas State University

  • Embedded Software Engineering Course, CCN 25754, Spring 2002, Department of Electrical Engineering and Computer Sciences, University of California at Berkeley

Additional Software related to brickOS

  • EmulegOS - C++ code you can compile and link together with your legOS code. The resulting application emulates the behavior of your code as it were running on the actual RCX.
