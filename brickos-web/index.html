The brickOS Homepage

brickOS is the new name for the project formerly known as legOS.

So what is brickOS?

brickOS is an Open Source project that provides a C and C++ programming environment for the Lego Mindstorms Robotics Kits, allowing the owner of such a kit to program in good old-fashioned C (or C++) instead of the standard Lego Programming Language. It was originally developed on Linux, but should run on most Unices as well as Windows.

Docs

All brickOS documentation is now linked to from the docs page here at Sourceforge. There, you'll find links to the HOWTO, API docs, and everything else that you'll need to get started working with brickOS.
