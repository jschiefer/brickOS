So what is brickOS?

brickOS is an Open Source project that provides a C and C++ programming environment for the Lego Mindstorms Robotics Kits, allowing the owner of such a kit to program in good old-fashioned C (and now C++) instead of the standard Lego Programming Language. It was originally developed on Linux, but should run on most Unices as well as Windows.

brickOS consists of an alternative operating system for the Mindstorms RCX and demonstration programs written in C and C++. Also provided are utilities which allow you to download the OS to the RCX and download your compiled programs to the RCX.

The brickOS distribution provides the sources for the operating system, the demo programs and the utilities. You will also need an Hitachi H8 cross compiler assembler and linker (basic toolchain). Our instructions, referred to in the following Docs section, help you obtain and configure these tools. After they are installed you will be able to build the OS, the utilities and finally, programs for controlling your own creations.
