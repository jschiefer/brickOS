Welcome to the brickOS home page!

(brickOS is our new name for the legOS project)

brickOS is an alternative operating system for the Lego Mindstorms RCX Controller.  It also provides a C/C++ development environment for RCX programs using gcc and g++ (the GNU C and C++ cross compilation tool chain) and the necessary tools to download programs to the RCX. If you program in C or C++ and would like to be able to write programs for your RCX in these same languages than brickOS is for you.

We encourage everyone to contribute changes and help with testing of brickOS, and we provide access to our development sources with anonymous CVS and test releases.

We are trying to provide a couple of releases each year and are always looking for further ideas and enhancements for brickOS.

News/Announcements

The brickOS project is in transition from our original name legOS to our new name brickOS. During this transition you will find links to our prior site (links with legOS name in them) and install files still being distributed using our prior name. It will take some time for this all to change to our new name. So, we ask for your patience while we make this change.
Thank you. -brickOS Team
