Test versions of our next brickOS release

Last update: 05 Sep 2002 21:25

This page provides access to our development releases of brickOS.

At this time we are working on:

  • Our next major release called brickos-1.0.0
    • Universal Remote Control support
    • Simple RCX to RCX messaging via IR
    • Up to date API docs, HOWTO, etc.
    • New third party sensor support - details soon
    • Add USB IR Tower support for Linux  NEW!
    • More demo code/examples
    • ...and more?...

We've just completed:

  • A tiny release called legOS-0.2.6
    • USB IR Tower support for Windows (only)
    • Temperature Sensor support for C++
    • Latest changes in CVS since 0.2.5
    • Bug fixes, and more see: NEWS, ChangeLog, and README
    • (We are not intending to announce this tiny release wider than the legOS list on LUGNET)

Feedback

We are offering release candidates in order to get more testing against them than we can do by ourselves. This means that we need your testing results as soon as you can report them. Likewise our goal is to address what you are seeing as quickly as possible and get the next release candidate out. It is also useful for you to tell us it works for the platform and compiler tools you were using (of course, tell us what these are ;-)

Reporting what you find

observations without fixes
Simply post these to legOS at LUGNET as responses to the appropriate announcement post
 
you've attempted the fix and here's the patch
View and post Patches at brickOS project site Select "submit new" (when you get there) to submit a new patch
 
you found a bug that doesn't appear to be related to this test release
View and post Bugs at brickOS project site Select "submit new" (when you get there) to submit a new bug report
 
you would like to file an enhancement request
View and post Enhancement requests at brickOS project site Select "submit new" (when you get there) to submit a new enhancement request
